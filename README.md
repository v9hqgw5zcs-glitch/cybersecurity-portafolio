SOC Analyst Lab Portfolio

Cybersecurity Analyst in Training  

 Overview

This repository documents my hands-on cybersecurity labs focused on building practical skills in network reconnaissance, enumeration, and threat analysis.

Each lab is structured to simulate real-world workflows used by SOC analysts and security professionals.

 Core Skills Developed

- Network Scanning (Nmap)
- Host Discovery & Enumeration
- Port Scanning & Analysis
- Understanding Network Security Controls

 Objective

To develop real world cybersecurity skills through consistent, hands on lab work and documentation, with the goal of transitioning into a SOC Analyst role.

 Disclaimer

All activities in this repository were conducted in controlled, ethical lab environments for educational purposes only
